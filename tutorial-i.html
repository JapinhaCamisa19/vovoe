Como Baixar o INSTAGRAM

  1. Abra o PlayStore: Procure o ícone da Play Store no seu celular e toque nele.
  2. Pesquise o aplicativo: Toque na barra de busca e digite o nome do app que deseja, como instagram.

INSTALAR O INSTAGRAM

  1. Toque em “Instalar”: Quando aparecer o instagram, toque no botão azul com o nome Instalar.
  2. Espere a instalação: Aguarde até o processo terminar. Quando o botão mudar para Abrir, o app estará pronto.
  3. Abra o aplicativo: Clique em Abrir para iniciar o instagram e começar o uso.

CRIAR CONTA

  1. Na tela inicial de login você verá os campos para inserir o nome de usuário e senha.
  2. Ignore os campos de login se você ainda não tem uma conta.
  3. Role a tela para baixo até encontrar a opção "Criar nova conta".
  4. Clique no botão "Criar nova conta" (ele está destacado com uma seta vermelha apontando para ele na imagem).
